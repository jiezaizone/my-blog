## Isuwang技术部日常技术分享

## Build Setup


# 安装
$ npm install

# 开发模式需切换到dev分支下开发
# 启动服务
$ hexo server
或者
$ hexo s

默认情况下，访问网址为： http://localhost:4000/

### 部署
1) 执行以下命令生成博客的静态页面
$ hexo generate
或者 $ hexo g

2) 执行以下命令将我们生成的博客静态页面上传到GitHub
$ hexo deploy
或者 $ hexo d

3) 打开浏览器访问 username.github.io 即可访问我们刚部署到Github上的博客
